# Frankston 2035 — Art That Lives
## Council Pitch Document

*For: Frankston City Council officers and Councillors*
*Prepared by: Blue Snake Studio*
*Date: June 2026*

---

## In One Sentence

Frankston 2035 is a ten-year creative infrastructure program that puts permanent art in schools, healthy technology in libraries, and self-guided creative walks in the city's streets — starting with three fully scoped pilots.

---

## The Problem

Frankston has creative people. It lacks the infrastructure to connect them.

- Young people in Frankston have fewer publicly funded creative opportunities than comparable Victorian cities.
- Technology in schools and libraries increasingly extracts children's data and attention, without returning lasting creative value.
- Public art exists in Frankston — but it does not teach, it does not connect, and it does not grow.

This is not a criticism. It is a structural gap that other cities have closed. Frankston can close it too.

---

## The Opportunity

Three forces are converging:

**Cultural:** Post-pandemic demand for place-based meaning. Public art investment is resurging nationally. Communities want to see themselves reflected in their city.

**Technological:** Growing concern about children's digital safety. Parents, teachers, and councils are actively looking for alternatives to platform-based technology in learning environments.

**Funding:** Creative Victoria, Australia Council, and federal place-based investment programs are actively seeking proposals in exactly this space. The window is open.

---

## The Three Pilots

Three concrete, costed, permanent outcomes. No events. No reports. No consultancies that leave nothing behind.

### Pilot 01 — The Living Mural
A curriculum-linked mural co-designed with students in a Frankston primary school.

| | |
|---|---|
| **Where** | A Frankston primary school (site to be confirmed with Council) |
| **Timeline** | 12 months from commission |
| **What's permanent** | Large-scale mural + curriculum resource pack owned by the school |
| **Budget** | $45,000–$65,000 |
| **Council role** | Facilitate school relationship + permit support |
| **Who owns it** | The school, permanently |

The mural is not decoration. It is built around curriculum content. Students co-design it. Teachers use it. The artist is credited. The wall teaches long after BSS has left.

---

### Pilot 02 — The Healthy Tech Library
A dedicated creative technology space in Frankston Library — no accounts, no tracking, no advertising.

| | |
|---|---|
| **Where** | Frankston Library |
| **Timeline** | 6 months setup + ongoing programming |
| **What's permanent** | Installed space + trained facilitators + ongoing program owned by the Library |
| **Budget** | $30,000–$50,000 |
| **Council role** | Facilitate library partnership |
| **Who owns it** | Frankston Library Service, permanently |

Every tool meets the Healthy Technology Charter. Drawing. Music. Writing. Building. No child creates an account. The screen is a doorway, not a destination.

---

### Pilot 03 — The QR Art Walk
A self-guided creative walk through Frankston using QR codes to surface local stories, artworks, and audio.

| | |
|---|---|
| **Where** | Frankston CBD and foreshore |
| **Timeline** | 6 months development + permanent installation |
| **What's permanent** | 10–15 QR stops + printed map + expandable web system |
| **Budget** | $20,000–$35,000 |
| **Council role** | Site access + signage permits |
| **Who owns it** | The city and community, permanently |

No app. No download. Content created by local artists and community members. New stops can be added by anyone following the system. It is owned by Frankston, not by a platform.

## Why Start With the Library Creative Pack

The recommended first pilot is a scaled version of Pilot 02: a curated set of offline creative tools installed in Frankston Library.

**Cost:** $20,000–$30,000 total. Council contribution: $12,000–$15,000.
**Timeline:** 3–4 months from decision to open.
**Why:** Single partner. No software development. No First Nations cultural content requirements. Visible, permanent outcome. Demonstrates the project's core philosophy immediately. Builds the Library relationship before asking for more.

Full comparison of all five pilot options is in `PILOT-DECISION-SHEET.md`.

## What Protects This Project

**First Nations protocol:** A documented 9-part process governing all engagement with First Nations community. The process begins with listening, not content creation. No cultural material is used without explicit written permission. Full protocol is an internal working document pending review by an independent cultural safety advisor.

**Healthy Technology Charter:** Three-question test applied to every technology used near children. Fail any question: the technology is rejected.

**Risk register:** Full risk tracking across strategic, operational, cultural, financial, and pitch-stage risks. Current register at `../05-risk-register.md`.
