## PILOT 01 — THE LIVING MURAL

**A curriculum-linked mural, designed with students, permanent on a school wall.**

| | |
|---|---|
| **Where** | A Frankston primary school (site TBC with Council) |
| **When** | 12 months from commission to installation |
| **Lead partner** | School principal + selected artist |
| **What's permanent** | One large-scale mural + one curriculum resource pack |
| **Budget range** | $45,000–$65,000 |
| **What Council does** | Facilitates school relationship + permit support |
| **What BSS does** | Commissions artist, runs co-design, produces curriculum resources |
| **What the school gets** | Artwork owned permanently. Resources free. Artist credited. |

**The difference from a normal mural commission:** Students co-design it. Teachers use it in class. The curriculum link is built into the visual content. The artist is a temporary educator. The wall teaches.

## PILOT 02 — THE HEALTHY TECH LIBRARY

**A dedicated creative technology space in Frankston Library — no accounts, no tracking, no advertising.**

| | |
|---|---|
| **Where** | Frankston Library |
| **When** | 6 months setup + ongoing programming |
| **Lead partner** | Frankston Library Service |
| **What's permanent** | Installed space + trained facilitators + ongoing program |
| **Budget range** | $30,000–$50,000 |
| **What Council does** | Facilitates library partnership |
| **What BSS does** | Designs space, evaluates tools, trains facilitators, produces parent guide |
| **What the library gets** | A running program it owns. No ongoing BSS dependency. |

**The difference from a computer lab:** Every tool meets the Healthy Technology Charter. No child creates an account. The screen is a doorway, not a destination. The library owns and runs it after launch.

## PILOT 03 — THE QR ART WALK

**A self-guided creative walk through Frankston, using QR codes to open local stories, artworks, and audio.**

| | |
|---|---|
| **Where** | Frankston CBD and foreshore |
| **When** | 6 months development + permanent installation |
| **Lead partner** | Community (sites nominated by community) |
| **What's permanent** | 10–15 QR stops + printed map + expandable web system |
| **Budget range** | $20,000–$35,000 |
| **What Council does** | Facilitates access to public sites + signage permits |
| **What BSS does** | Designs system, commissions content, builds static web pages, produces map |
| **What the community gets** | A living walk they can add to. No app. No account. No tracking. |

**The difference from a heritage trail:** The content is created by local artists and community members. It grows over time. A new QR stop can be added by anyone following the system. It is owned by the city, not by a platform.

## What All Three Have in Common

Every pilot is designed to:

- **Produce a permanent outcome** — not an event, not a report, not a memory
- **Transfer ownership** to the community institution on completion
- **Generate a replication guide** so another school, library, or city can copy it
- **Leave nothing behind that requires BSS to maintain**
