## Design System Reference

All print assets must conform to the Publication Design Guide (`../09-publication-design.md`).

**Quick reference:**
- **Display type:** Canela Light / Canela Regular
- **Body type:** Neue Haas Grotesk
- **Data / labels:** IBM Plex Mono
- **Palette:** Deep Ink `#1A1A1A` · Warm White `#F8F6F1` · Frankston Blue `#1B4F8A` · Coastal Sand `#D4C4A0`
- **Grid:** 12-column, A4, 15mm outer margins, 10mm inner margins
- **Photo:** Documentary, natural light. No stock photography.

## Print Production Notes

- **Preferred: print on demand** for initial runs. Offset only when run size exceeds 500 units.
- **Paper:** 120gsm uncoated for all text documents. 350gsm for posters and covers.
- **Colour:** CMYK conversions from hex values specified in `../09-publication-design.md`.
- **Accessibility:** All text must meet WCAG 2.1 AA contrast ratios. No text below 9pt in print.
