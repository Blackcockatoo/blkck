# B$S Proof Strip System

Every main Moss Tree branch now includes a four-part proof strip.

## Pattern

1. **What it is** — the plain identity of the branch.
2. **Why it matters** — the reason someone should care.
3. **Exists already** — proof that this is built or underway.
4. **Next move** — the action that turns interest into momentum.

## Rule

Spell on top. Tool underneath.

The proof strip is designed to stop visitors getting lost in the mythic style. It gives every page a short credibility layer without killing the B$S atmosphere.
