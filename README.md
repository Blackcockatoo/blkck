## GitHub Pages

To publish this as a page:

1. Open the repository settings.
2. Go to **Pages**.
3. Set source to **Deploy from a branch**.
4. Choose `main` and `/root`.
5. Save.

The page should then serve from the repository’s GitHub Pages URL.
